# Login System - Web

Implementation: 
Set up your database configuration (database, user, password and dialect) on loginSystem.js
Set up html pages
Run node loginSystem.js


## tools used
- Node
- Bulma
- express
- bodyParser
- sequelize
- express session
- cookie parser

## client 
access localhost:3000

## server
runs on localhost:3000
to start: node server.js
